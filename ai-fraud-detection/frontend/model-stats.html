How the Model Works
Random Forest
100 decision trees vote on whether a transaction is fraudulent. Handles class imbalance with SMOTE oversampling.
Isolation Forest
Anomaly detection layer that flags statistically unusual transactions independently of labels.
SMOTE
Synthetic Minority Oversampling balances the 0.17% fraud rate so the model learns fraud patterns effectively.
Combined Score
Final score = ML probability × 70 + Rule score × 30. Flagged as fraud if ≥ 40.
